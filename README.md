# End Semester Project – Multi Page Website

##  Description
This is a multi-page web application built using HTML, CSS, and JavaScript.  
It follows a structured user flow starting from a landing page, moving through login, and then navigating to multiple feature pages.

---

##  How to Run

1. Download or clone the repository  
2. Open **first_page.html** in your browser  
3. Navigate through the project using buttons and links  

---

##  Project Flow

First Page → Login Page → Main Page → Feature Pages

---

##  Key Files

- first_page.html → Landing / Starting Page  
- login_page.html → Login Page  
- main.html → Main Dashboard  

---

##  Features

- Multi-page navigation  
- Structured user flow (Landing → Login → Dashboard)  
- Interactive UI with images and styling  
- JavaScript functionality   
- Clean layout with responsive design elements  

---

##  Technologies Used

- HTML  
- CSS  
- JavaScript  

---

##  Note

- Start the project by opening **first_page.html**  
- Login is frontend-based (no backend validation required)  
- Use navigation links/buttons to explore all pages
